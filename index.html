1) Evolutionary Trends in Distributed and Cloud Computing

  1. Evolution TimeLine:

    2. Mainframe Computing:

    • They are responsible for handling large data such as input-output opearation.
    • Mainframe computing are very expensive.

    Distributed Systems:

    • Distributed Systems is the composition of multiple independent systems
    • The Main purpose of Distributed System is to share the Resources and use them effectively and efficiently

    Cluster Computing:

    • Cluster Computing came as an alternative to the Mainframe Computing.
    • Cluster Computing computing are very cheap.

    Grid Computing:

    • Different Machines are placed at different geographical locations and are connected via the Internet
    • Cloud Computing is called as the Successor of Grid Computing.

    Cloud Computing:

    • Cloud Computing means storing and accessing the data and program by using Internet.
    • Cloud Computing is also called as the Internet-Based Computing.

    IoT & CPS:

    • IoT : Internet of Things.
    • CPS : Cyber Pysical Systems.
    • IoT and CPS connects real world devices to the cloud systems by using internet.
  2. HPC to HTC:
    • HPC : High Performance Computing.
    • HTC : High Throughput Computing.
    • Early Computing focuses on HPC for performing the tasks. But as Cloud Computing and Web applications grew, The Focus is Shifted to the HTC for performing the tasks.
  3. Cloud Services:
    • Cloud Services are divided in to three categories:
      • IaaS - Infrastructure as a Service
      • PaaS - Platform as a Service
      • SaaS - Software as a Service
